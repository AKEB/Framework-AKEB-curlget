# CurlGet

## Install

composer project akeb/curlget

Composer config

```json
{
    "require": {
        "akeb/curlget": "^1.1.0"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/AKEB/Framework-AKEB-curlget"
        }
    ]
}
```

or

```bash
composer require akeb/curlget
```
